smpl_doc.doc.append

smpl_doc.doc.append(txt)[source]

Append txt in the target function docstring.

Examples

>>> @append('hi')
... def ho(): pass
>>> ho.__doc__
'hi'
>>> @append(ho)
... def hi(): pass
>>> hi.__doc__
