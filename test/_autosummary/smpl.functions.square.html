smpl.functions.square

smpl.functions.square(x, x_0, a, y)[source]

square(x,x_0,a,y) = \(a \left(x - x_{0}\right)^{2} + y\)

>>> from smpl.functions import square
>>> from smpl import plot
>>> plot.function(square,1,5,0,xmin=-5,xmax=5)
