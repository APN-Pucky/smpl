Parallel evaluation of the list generator from gen().

Return parallel executed values.

Examples¶

>>> def twice(x):
...     return x+x
>>> for r in [calc(twice,i) for i in range(0,5)]:
...     print(res(r))
0
2
4
6
8
>>> res([calc(lambda x : x**3, i) for i in range(0,5)])
[0, 1, 8, 27, 64]
