smpl.functions¶

A list of various functions

const(x, m)

cos(x,a,f,phi) = \(a\cos(2πf(x-\phi))\)

cos_abs(x, a, f, phi)

cos_abs(x,a,f,phi) = \(a|\cos(2πf(x-\phi))|\)

cube(x,a,b,c,d) = \(ax^3+bx^2+cx+d\)

exp(x, c, y_0)

exponential(x,c,y_0) = \(\exp(cx)y_0\)

gauss(x, x_0, A, d, y)

gauss(x,x_0,A,d,y) = \(A\cdot \exp\left(\frac{-(x-x_0)^2}{2d^2}\right)+y\)

line(x, a, b)

line(x,a,b) = ax+b

linear(x, m)

linear(x,m) = mx

log(x, c, y_0)

log(x,c,y_0) = \(\log(cx)y_0\)

lorentz(x, x_0, A, d, y)

lorentz(x,x_0,A,d,y) = \(\frac{A}{\pi d (1+ (\frac{x-x_0}{d})^2)} + y\)

sin(x, a, f, phi)

sin(x,a,f,phi) = \(a\sin(2πf(x-\phi))\)

sqrt(x, a, b, c)

sqrt(x,a,b,c) = \(a\sqrt{x+b}+c\)

square(x, x_0, A, y)

square(x,x_0,A,y) = \(A(x-x_0)^2+y\)

tan(x,a,f,phi) = \(\tan(2πf(x-\phi))a\)
