smpl.functions¶

A list of various functions

const(x, m)

const(x,m) = m

cos_abs(x, a, f, phi)

cos_abs(x,a,f,phi) = \(a|\cos(2πf(x-\phi))|\)

exponential(x, c, y_0)

exponential(x,c,y_0) = \(\exp(cx)y_0\)

gauss(x, x_0, A, d, y)

gauss(x,x_0,A,d,y) = \(A\cdot \exp\left(\frac{-(x-x_0)^2}{2d^2}\right)+y\)

line(x, a, b)

line(x,a,b) = ax+b

linear(x, m)

linear(x,m) = mx

lorentz(x, x_0, A, d, y)

lorentz(x,x_0,A,d,y) = \(\frac{A}{\pi d (1+ (\frac{x-x_0}{d})^2)} + y\)

quadratic(x, x_0, A, y)

quadratic(x,x_0,A,y) = \(A(x-x_0)^2+y\)
