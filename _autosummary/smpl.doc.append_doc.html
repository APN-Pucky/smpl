smpl.doc.append_doc¶

smpl.doc.append_doc(original)[source]¶

Append doc string of original to target object.

Parameters¶

originalclass or function

orignal.__doc__ is appended to the __doc__ of the target

Examples¶

>>> def ho():
...     '''Ho'''
...     print(ho.__doc__)
>>> @append_doc(ho)
... def hi():
...     '''Hi'''
...     print(hi.__doc__)
>>> hi()
HiHo
